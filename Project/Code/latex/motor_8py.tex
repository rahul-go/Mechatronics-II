\subsection{Detailed Description}
The motor module contains the Motor\+Driver class. 

The Motor\+Driver class implements a motor driver for the M\+E405 board. At any point while the motor is running, the user can use this class to modify the duty cycle of the signal and the motor will react appropriately.
