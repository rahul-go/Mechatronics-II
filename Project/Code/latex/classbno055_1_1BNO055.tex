\subsection{Detailed Description}
\begin{DoxyVerb}Driver for the BNO055 9DOF IMU sensor.

Example::

    import bno055
    from machine import I2C, Pin

    i2c = I2C(-1, Pin(5), Pin(4), timeout=1000)
    s = bno055.BNO055(i2c)
    print(s.temperature())
    print(s.euler())
\end{DoxyVerb}
